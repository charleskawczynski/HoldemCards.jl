PlayingCards.Suit — Type
Suit

Subtypes are used for each card suit (all of which have aliases):

  • Club (alias ♣)
  • Spade (alias ♠)
  • Heart (alias ♡)
  • Diamond (alias ♢)

PlayingCards.Rank — Type
Rank

The card rank, subtypes are used for each card rank including

  • Ace
  • King
  • Queen
  • Jack
  • NumberCard{N} where 2 ≤ N ≤ 10
source
PlayingCards.Card — Type
Card{R <: Rank, S <: Suit}

A playing card. Can be constructed with

Card(rank, suit), or by convenience constructors. For example:

  • 2♢ (equivalent to Card(NumberCard(2), Diamond()))
  • A♡ (equivalent to Card(Ace, Heart()))

A 10-suit can be constructed with one of two constructors:

  • 10♣ (equivalent to Card(NumberCard(10), Club()))

or

  • T♠ (equivalent to Card(NumberCard(10), Spade()))
